Projects

Echo Project

Discord-like app made open-source using Electron and React!

Our goal is to bring the simplicity of Discord, and merge it with the self-hosted nature of TeamSpeak, while keeping everything open-source.

Clipmaner

Clipmaner is a website made using NextJS and React to let streamers download their content easly and in bulk

The project is open source and is hosted on clipmaner.com

GuessTheSong

GuessTheSong is a Proof of concept for a minigame website. The goal was to make a pleasing and modern for the eyes websitte where players had to guess the song that was playing. The fastest would earn points that would have been counted at the end of a match

The project is open source but it never became a thing since the decision of moving everything to SSR with NextJS, forcing then a rewrite of most of the code.

zTh_Phone

zTh_Phone is a resource that I made for a GTA5 mod, called FiveM.

This mod for the game makes modding GTA5 simple, expetially for multiplayer, and it's very famous for it's RP community

I made this mod almost 4 years ago, and it's purpouse was to emulate an actual Phone, so that players can communicate and use different apps.

The project is long dead, but this helped me learn about VUEjs and JS web in general, both client and server side.
